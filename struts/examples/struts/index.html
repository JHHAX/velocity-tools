Velocity for Struts

Version 0.6, March 5. 2002		  

This is a web application to demonstrate the integration of Velocity with Struts. It contains basic documentation and serveral example applications.

This includes version 0.6 of Velocity for Struts. The API has been completely redesigned and is now more complete and more systematic. Reference documentation for the included context tools has been added.

Highlights
  • Velocity is now becoming an alternative view technology for Struts-based Web applications.
  • Velocity templates can be mixed with JSP templates in the same application.
  • No changes are required to Struts. This works with Struts 1.0, 1.01, 1.02 and the upcoming Struts 1.1.
Overview
  • Example Applications
    Five small example application have been included. Look at example 2 for a demonstration of the Velocity context tools for Struts. JSP versions of some of the examples have been included to allow a direct comparison of the two view technologies.
  • Reference Documentation
    This is the reference documentation of the context tools.
    • MessageTool - handles internationalized messages
    • ErrorsTool - handles error messages
    • FormTool - provides support for form handling
    • LinkTool - works with URIs
  • JavaDoc
