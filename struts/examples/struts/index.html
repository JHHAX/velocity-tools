Velocity for Struts

Version 0.7, April 2. 2002		  

This is an example web application demonstrating the integration of the Struts web application framework with the Velocity template engine.

Highlights
  • Velocity is now becoming an alternative view technology for Struts-based Web applications.
  • Velocity templates can be mixed with JSP templates in the same application.
  • No changes are required to Struts. This works with Struts 1.0, 1.01, 1.02 and the upcoming Struts 1.1.
Status

The API has been redesigned completely and is now more complete and more systematic. An early draft of several documentation pages is now available.

This is an unrealeased work in progress. New versions will be announced on the Velocity mailing list and home page.

Application Examples

Five small application examples have been included. Look at example 2 for a demonstration of the Velocity context tools for Struts. JSP versions of some of the examples have been included to allow a direct comparison of the two view technologies.

Documentation

An early draft of the documentation is available online. This is a work in progress but I guess it is better than no documentation at all.
